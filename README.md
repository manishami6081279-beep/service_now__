# Automating Employee Data

## Project Description
This project demonstrates how to automate employee data creation using ServiceNow.

## Steps Involved
1. Created employee data in Excel
2. Imported data into ServiceNow
3. Created Transform Map
4. Mapped fields using Mapping Assist
5. Transformed data into target table

## Tools Used
- ServiceNow
- Excel

## Output
Employee records successfully created in ServiceNow.
